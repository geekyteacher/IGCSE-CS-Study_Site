Databases

A database is an organised collection of data, which allows users to obtain and process information according to their requirements. Databases can be an efficent way of storing data, helping avoid duplicaiton and making searching easier.

Databases store data in tables. There are two types of database

  • a flat-file database: this contains a single table
  • a relational database: built of multiple tables that link together
Databases are used for a wide range of diifferent purposes for example address books, school records, financial records.
